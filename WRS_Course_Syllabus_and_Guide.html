Part 1 — What this course is

This Canvas course is the instructional backbone for Nevada's Workplace Readiness Skills (WRS) standards — the 21 assessed competencies that CTE students demonstrate at the end of their programs. It is designed to be used across program years, across pathways, and across classroom formats without modification.

The course does not replace your CTE pathway instruction. It supplements it — giving you ready-made, standards-aligned modules you can deploy as openers, closers, full lessons, or sub plans, depending on where your students are and what you need on a given day.

Assessment alignment: Every module in this course is explicitly aligned to the WRS assessment that students take at the end of their CTE program. The knowledge checks, scenario questions, and discussion prompts are designed to build the vocabulary, judgment, and situational awareness students need to perform well on that assessment — not just to check a curriculum box.

Part 2 — Course structure

One course, not two

This course is intentionally unified. There is no separate Year 1 and Year 2 version. The WRS assessment covers the same 21 standards regardless of where a student is in their program — so the instructional resource mirrors that structure. You decide which modules to assign and when, based on where your students are.

Two depth levels per standard

Every standard has content at two depth levels:

  • Foundational — introduces the standard, defines key terms, uses accessible scenarios, builds basic recognition and comprehension. Appropriate for students early in a program or encountering a standard for the first time.
  • Applied — assumes familiarity with the standard and pushes into harder situations, judgment calls, and exam-style scenario questions. Appropriate for students preparing for the WRS assessment or needing a deeper challenge.

These are not year-level designations. A second-year student may need Foundational content on a standard they've never seen. A strong first-year student may be ready for Applied on day one. You make that call.

Five modules per standard

Each of the 21 standards has five total modules — two tiers at Foundational depth, two tiers at Applied depth, and one Sub Lesson that draws from both:

Tier 1
Short Module
8–12 minutes

Opener, closer, or bell ringer. Self-contained. One scenario, 2–3 knowledge check questions, one exit ticket prompt. Works as a standalone activity or a warm-up before a longer lesson.

Tier 2
Extended Module
45–60 minutes

Full block lesson. Structured instruction, interactive activity, 5–6 question quiz with per-answer feedback, discussion exit ticket. Designed to fill a complete class period with minimal teacher prep.

Tier 3
Sub Lesson
75–90 minutes

Everything in the Extended Module plus an anchoring video, additional scenario content, extended discussion structure, and explicit sub-facing facilitation instructions. No teacher prep required — a sub can run this cold.

That structure — 21 standards × 5 modules each — produces 105 total modules across the course. You will never use all of them in a single year, and you are not expected to. This is a library. Use what fits.

Foundational + Applied

Foundational modules come first — Short then Extended — followed by Applied in the same order. The Sub Lesson draws from both depth levels and sits at the end as a standalone coverage tool. You do not have to follow this sequence. Use what fits.

Suggested pacing

  • Daily bell ringer: Rotate through Short Modules across standards throughout the year. At 3–4 per week you'll cover all 21 standards multiple times.
  • Monthly WRS day: Dedicate one class period per month to an Extended Module. You'll cover all 21 standards across a school year with room to revisit industry-priority standards.
  • Pre-assessment push: In the 4–6 weeks before the WRS assessment, run Applied Short Modules daily as exam prep warm-ups.
  • Sub coverage: Drop in a Sub Lesson any time you're out. No prep required — instructions are built into the module.

Industry-priority standards — start here

If you're new to this course or introducing it to students for the first time, start with the four industry-priority standards. Nevada employers identified these as the most common gaps in new hire readiness:

  • 1.1.1 Positive Work Ethic
  • 1.1.3 Teamwork
  • 1.1.4 Self-Representation
  • 1.2.1 Speaking & Listening

Part 5 — Grading and credit: your options

All modules in this course are published with discussions and exit tickets set to ungraded by default. This is intentional — every teacher's gradebook is structured differently, and no point value should be baked in that you then have to undo.

You have three options for how to handle credit. Choose one and apply it consistently, or mix approaches across standards.

Option A
Grade individual discussions on a per-module basis
Turn any discussion into a graded assignment directly in Canvas. Best for teachers who want granular tracking of student engagement across standards.
  1. Open the Discussion in Canvas
  2. Click Edit
  3. Check the Graded box
  4. Set your point value (5–10 pts recommended for a discussion exit ticket)
  5. Add a due date if needed
  6. Click Save
Best for: Teachers who assign WRS modules regularly and want them reflected in the gradebook. Works well with a simple completion rubric: full points for a substantive post + one reply, partial for post only, zero for nothing.
Option B
Use a single external assignment as a reflection log
Create one Canvas Assignment — something like WRS Reflection Log — worth a set number of points. Students submit written responses from multiple module exit tickets into that one assignment over time as a running document. You grade it periodically or at the end of a grading period.
  1. Go to Assignments → + Assignment
  2. Name it: WRS Reflection Log
  3. Set points to whatever fits your gradebook (50–100 pts for a semester-long log)
  4. Set submission type to Text Entry or File Upload
  5. In the instructions, tell students to paste their exit ticket responses here as they complete modules throughout the semester
Best for: Teachers who use WRS modules as supplemental content and want one clean gradebook entry rather than a discussion grade for every standard. Reduces gradebook clutter significantly.
Option C
No Canvas grade — assess through the WRS assessment itself
Use the modules purely as instructional and exam prep resources. Don't grade anything in Canvas. The WRS assessment at the end of the program is the summative measure — the modules are the preparation for it.
Best for: Teachers who use WRS modules as bell ringers or quick reviews and don't want to add to their grading load. Also appropriate when WRS instruction is embedded in pathway content rather than treated as a standalone grade.
Note on the interactive quizzes: The knowledge check quizzes built into each module give students instant feedback and a score, but they do not report to the Canvas gradebook. They are formative — designed to help students self-assess and review before the WRS assessment, not to generate a grade. If you want quiz scores in your gradebook, build a parallel Canvas Quiz using the New Quizzes engine with the same questions.

Part 6 — Technical notes

How the modules work in Canvas

Each module is an HTML file hosted on GitHub Pages and embedded in a Canvas Page via iframe. This means:

  • All interactivity — quizzes, drag-and-drop activities, scenario accordions — works natively without any plugins or LTI tools
  • Modules work on desktop and mobile
  • No student login is required to access the interactive content
  • The URLs are stable — they will not change or break unless the GitHub repository is modified

If a module isn't displaying correctly

  • Try a different browser — Chrome is most reliable
  • Check that you're viewing the Canvas Page in student view, not edit mode
  • If the iframe shows a blank space, your institution's Canvas security settings may be blocking external iframes — contact your Canvas admin

Requesting updates or corrections

All modules are maintained centrally. If you find an error, a broken link, or content that needs updating, contact your CTE coordinator. Do not edit the Canvas Pages directly — changes need to be made at the source file level to stay consistent across the course.
